# Teslemetry TypeScript Monorepo

This repository contains the official TypeScript/JavaScript tools and libraries for interacting with [Teslemetry](https://teslemetry.com).

## Packages

### [@teslemetry/api](./packages/api/README.md)

The core TypeScript/JavaScript client library for the Teslemetry API. Provides full control over vehicles, energy sites, and real-time streaming (SSE).
